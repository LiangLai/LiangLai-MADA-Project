# Tables

This folder contains tables generated by the analysis scripts.

Key outputs include baseline characteristic tables, Cox model estimates, proportional hazards diagnostics, landmark sample summaries, and predictive ML model-comparison tables.

ML tables include:
- `ml_w3_sample_summary.csv`
- `ml_cv_performance.csv`
- `ml_test_performance.csv`
- `ml_best_lasso_params.csv`
- `ml_best_rf_params.csv`
- `ml_rf_variable_importance.csv`
