# Figures

This folder contains figures generated by the processing, exploratory analysis, and modeling scripts.

Key outputs include exploratory figures for sample size, cognitive status, dementia prevalence, BMI variability distributions, Cox landmark trend figures, and predictive ML model-comparison figures.

ML figures used in the final manuscript include:
- `ml_model_comparison.png`
- `ml_roc_curves.png`
- `ml_pr_curves.png`
- `ml_rf_variable_importance.png`
